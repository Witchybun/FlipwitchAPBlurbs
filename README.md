# FlipwitchAPBlurbs
Repository for the "blurbs" that show up on items upon collecting them in the randomizer.  This is meant to go alongside the graphic repository [KaitoKid has made for Stardew Valley](https://github.com/agilbert1412/ArchipelagoUtilities/tree/main/KaitoKid.ArchipelagoUtilities.Net/KaitoKid.ArchipelagoUtilities.AssetDownloader/Assets), which this game also supports.

## Format
Submissions for formats are currently as follows:

- The game has an entry on KaitoKid's repository.
- Less than 100 characters.
- Has a similar flair or style to the original game (a little silly, possibly relate the item as if it was in the actual game for a moment in time)

Name and description format is that it follows that game's `Alias` system.  For example, Lunacid [has a aliases.json in KaitoKid's repository](https://github.com/agilbert1412/ArchipelagoUtilities/blob/main/KaitoKid.ArchipelagoUtilities.Net/KaitoKid.ArchipelagoUtilities.AssetDownloader/Assets/Lunacid/aliases.json).  You can either:

- Give a description to a specific item anyway
- Give a description to the alias, which applies to every item in the alias registry.

So in this example one could make a blurb for the `Material` alias for every Lunacid item that was a material (I don't do this since its more fun to work with every particular item).

There's a few examples already in this repository for the sort of style I'm looking for.  Though ultimately I can decide whether or not to merge the request.
